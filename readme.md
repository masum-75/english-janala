# English Janala

English Janala is a simple and interactive English learning web application.  
Users can explore lessons, learn new vocabulary, see meanings, sentences, and synonyms.



## Live Demo

Live Site: https://english-janala-2026.netlify.app/



##  Features

-  Multiple Lesson Levels
-  Vocabulary with Meaning
-  Pronunciation Display
-  Example Sentences
-  Word Details Modal
-  Synonyms Display
-  Loading Spinner
-  FAQ Section
-  Fully Responsive Design


## Technologies Used

- HTML5
- CSS3
- Tailwind CSS
- DaisyUI
- JavaScript (ES6)
- Netlify (Deployment)
